# Money Tracker Bot (MVP)

Telegram-бот для учёта доходов, расходов, участников и будущих платежей.

## Стек

- Python 3.12+
- aiogram 3
- SQLite (sqlite3)
- python-dotenv

## Запуск локально

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Создать `.env` на основе `.env.example` и указать токен:

```env
BOT_TOKEN=...
```

3. Запустить бота:

```bash
python bot.py
```

При первом запуске автоматически создаётся файл базы `data/finance.db` и нужные таблицы.

## Функциональность MVP

- Главное меню: операции, участники, платежи, отчёт.
- Участники: добавить, список, удалить.
- Операции (FSM): участник → тип (доход/расход) → сумма → категория → комментарий.
- Платежи (FSM): участник → название → сумма → дата.
- Отчёт: доходы, расходы, баланс, разбивка по участникам, ближайшие платежи.

## Deploy на Render (без Docker)

- **Build Command:** `pip install -r requirements.txt`
- **Start Command:** `python bot.py`
