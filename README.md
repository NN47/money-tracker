# Money Tracker Bot

Личный Telegram-бот для учёта финансов одного пользователя: доходы, расходы, предстоящие и регулярные платежи.

## Стек

- Python 3.12+
- aiogram 3
- aiohttp webhook
- Neon PostgreSQL
- psycopg2
- Render Web Service

## Переменные окружения

```env
BOT_TOKEN=...
WEBHOOK_URL=https://your-service.onrender.com
DATABASE_URL=postgresql://...
PORT=10000
```

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

## Основные разделы

- 💼 Главный экран: доходы/расходы/баланс месяца и ближайшие неоплаченные платежи (10 дней).
- ➕ Доход: FSM (сумма → категория → дата → комментарий).
- ➖ Расход: FSM (сумма → категория → дата → комментарий).
- 🔁 Регулярные платежи: показывает все добавленные регулярные платежи и открывает вложенное меню с кнопкой «➕ Добавить платеж».
- 📅 Календарь: показывает календарь событий с отмеченными днями и деталями по выбранной дате.
- ➕ Добавить платеж: FSM (тип → название → сумма → день месяца → категория → комментарий).
- 📊 Отчёт: доходы/расходы/баланс месяца, последние 10 операций, ближайшие 10 платежей.

## База данных

Используются таблицы:
- `transactions`
- `scheduled_payments`
- `recurring_operations`

Таблица `participants` может оставаться в БД, но новый код её не использует.

## Deploy на Render

- **Build Command:** `pip install -r requirements.txt`
- **Start Command:** `python bot.py`
